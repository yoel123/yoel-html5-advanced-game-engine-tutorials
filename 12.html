yoel html5 2d game engine programming tut 12

drag and drop

in this tutorial i will show how use my game engine drag and drop feature

the finished project is in file 12)drag and drop

for this tut you only need to add one line of code to init method 


				extanded_entity_p.init = function()
				{
					if(!this.did_init)
					{
						ytrace("did init");
						//init drag handler
						y_entity_p.drag.call(this);
						this.did_init = true;
					
						
					}
				}//end init

thats it,now you can drag the entity (not tested on smartphone).
