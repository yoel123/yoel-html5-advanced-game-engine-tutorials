move to

in this tutorial i will show how use my game engine entity move_to method

the finished project is in file 14)move_to

move to makes an entity move to an xy position.
first we willl crate move_to_do method and add it to update method. 


				extanded_entity_p.update = function()
				{
					//do stuff when class created
					this.init();
					
					this.move_to_do();
					
					

					//call parent update func
					y_entity_p.update.call(this); 

				}//end update

				extanded_entity_p.move_to_do = function()
				{
					t={"x":250,"y":190}
					y_entity_p.move_to.call(this,t);
				}//end alpha_do

in move_to_do we first daclere a target object.
the target is where the entity will move to.
the target can be any other entity or object with x and y attributes.
then we call move to and set its target (in this case t)
