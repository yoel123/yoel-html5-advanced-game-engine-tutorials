			extanded_entity_p.alpha_do = function()
			{
			  y_entity_p.alpha.call(this,this.alpha);
			  if(this.alpha >0 )
			  {
				this.alpha -= 0.003;
			  }
			  if(this.alpha <=0 )
			  {
				this.alpha = 1;
			  }
			}//end alpha_do

y_entity_p.alpha.call(this,this.alpha);
this line sets the entitys alpha (transparency).
it sets it to this.alpha att value.

the first if checks if alpha attribute is bigger then 0
reduce alpha att by some val.
if its less then 0 make alpha att = 1 (make it visible again).
alpha value can be from 0-1.

now we will go over rotation_do 


			extanded_entity_p.rotation_do = function()
			{
				y_entity_p.rotate.call(this,this.rotation);
				if(this.rotation>180){this.rotation = 0}
				if(this.rotation<180){this.rotation++;}
				this.rotation++;
				
			}//end rotation_do

first we call y_entity class rotate method and set it to this entity rotation att .
then we check if rotation is biger then 180 degrees, if its true set rotation to 0. if rotation is smaller then 180 degrees increment rotation.
